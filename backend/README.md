## Quick start

Clone the project

```bash
git clone https://github.com/adanj27/Patitas-a-casa.git && cd backend
```

Run docker compose.

```bash
docker compose up
```

## Usage

See [/docs/api/Api.md](https://github.com/adanj27/Patitas-a-casa/tree/main/backend/docs/Api/Api.md) to view how to send requests.
